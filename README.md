# 게임 설명

> [!WARNING] **플레이 환경 안내**
> 해당 게임은 터미널 환경을 사용합니다.  
> 게임 화면이 충분히 출력될 수 있도록 터미널 크기를 늘리고 시작해주세요.

## 메인 메뉴
메인 메뉴에서는 레벨 선택 및 게임 종료를 선택할 수 있습니다.
- 게임 시작 : 레벨 선택 화면으로 이동합니다.
- 게임 종료 : 게임을 종료하고 기존 터미널 상태로 이동 또는 프로그램을 종료합니다.

## 레벨 선택
레벨을 선택할 수 있습니다.

각 레벨은 $(n \times 10) \times (n \times 10)$ 사이즈의 맵을 가지고 있습니다.(ex.레벨 3의 맵 사이즈는 30 x 30)  
- 해당 레벨을 클리어한 기록이 경우 텍스트가 **하얀색**으로 표기되며, 레벨 옆에 최고 기록이 표기됩니다.
- 해당 레벨을 클리어한 기록이 없는 경우 텍스트가 <span style="color:gray">**회색**</span>으로 표기됩니다.

레벨 선택 시 게임 플레이로 이동합니다.

## 게임 플레이

### 플레이 방법
방향키를 선택하여 플레이어 오브젝트를 이동 시킬 수 있습니다.  
최단 시간 내에 플레이어 오브젝트를 종료 지점으로 이동시켜보세요.

### 오브젝트 안내
🔳 : 플레이어 오브젝트  
⬜️ : 일반 타일  
⬛ : 장애물  
🟩 : 종료 지점

## 기능 요구사항 정의
R-01-01 : 메인 메뉴에서는 게임 시작, 게임 종료를 선택할 수 있다.  
R-01-02 : 메인 메뉴에서는 게임 시작 선택 시, 레벨 선택 기능으로 이동한다.  
R-01-03 : 메인 메뉴에서는 게임 종료 선택 시, 프로그램을 종료한다.  
R-01-04 : 게임 이름을 ANSI Shadow 스타일로 문구를 출력한다.  

R-02-01 : 레벨 선택에서는 10단계의 레벨을 선택할 수 있다.  
R-02-02 : 레벨을 선택하면 게임 플레이 기능으로 이동한다.  
R-02-03 : 각 레벨은 해결 이력이 있는 경우와 없는 경우를 색상을 다르게하여 구별한다.  
R-02-04 : 해결 이력이 있는 경우, 클리어 시간을 출력한다.  
R-02-05 : ESC를 누르면 메인 메뉴로 이동한다.  
R-02-06 : ESC를 누르면 메인 메뉴로 이동함을 고지한다.  

R-03-01 : 오브젝트 별로 색상을 다르게 구별한다.  
R-03-02 : 오브젝트 안내 문구를 출력한다.  
R-03-03 : 플레이 시간을 출력한다.  
R-03-04 : 게임 시작 시, 플레이 시간 측정을 시작하며 클리어 판정이 되면 시간 측정을 종료한다.  
R-03-05 : 플레이 시간 측정 종료 후 기존 기록과 비교하여 최고 기록 유무를 판정하고, 최고 기록인 경우 저장한다.(덮어쓴다)  
R-03-06 : 플레이어 오브젝트는 플레이어가 선택한 방향으로 이동한다.  
R-03-07 : 플레이어 오브젝트는 장애물 오브젝트 또는 맵 끝 부분에 도달할 때까지 멈추지 않고 이동한다.  
R-03-08 : 플레이어 오브젝트가 종료 지점에 도달하면 클리어 판정한다.  
R-03-09 : 게임 클리어 시, 클리어 안내 문구를 출력한다.  
R-03-10 : 게임 클리어 시, 방향키를 통한 오브젝트 이동을 불허한다.  
R-03-11 : 게임 클리어 시, 레벨 변경하기를 선택할 수 있다.  
R-03-12 : 레벨 변경하기 선택 시, 레벨 선택 기능으로 이동한다.  

## 비기능 요구사항 정의
